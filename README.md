# XCode
Новый, очень простой язык программирования на python
XCode 1.0 — язык программирования
Автор: Damirus Papirus
Версия: 1.0
Тип: Интерпретируемый, динамический язык

📝 Описание
XCode 1.0 — это простой интерпретируемый язык программирования, созданный для автоматизации задач, системного администрирования, работы с файлами, сетью и создания скриптов. Язык сочетает простой синтаксис с небольшим набором встроенных команд.

🎯 Особенности
Простой синтаксис — легко читать и писать

Динамическая типизация — переменные не требуют объявления типа

Функции — поддержка пользовательских функций с параметрами и возвратом значений

Условные операторы — if/else через выражения сравнения

Циклы — loop с условием (аналог while)

Работа с файлами — создание, чтение, запись, удаление, поиск

Сетевые возможности — ping, browse

Файловая система — создание/удаление папок, рекурсивный поиск

Ввод/вывод — say для вывода, inp для ввода

Модульность — open для подключения других скриптов

Примечание: для работы требуется Python 3.10
